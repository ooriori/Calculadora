# Calculadora Básica en Rust

Este proyecto es una calculadora simple escrita en Rust que suma dos números fijos y muestra el resultado.

## Función principal

```rust
fn main() {
    let a = 5;
    let b = 3;
    let result = a + b;

    println!("la suma de {} + {} es: {} ", a, b, result);
}
